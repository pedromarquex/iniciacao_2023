## Bootstrap

### Install dependencies

Install the openbabel dependency on your system

```shell
sudo apt-get openbabel
```

Create a virtualenv and install the python dependencies

```shell
pip install -r requirements.txt
```

## Running the project

```shell
uvicorn server:app --reload
```
